## 🧠 What It Does

This workflow turns your Telegram bot into a **LinkedIn outreach engine**. You simply send a natural language query like `"Find 10 CEOs in Dubai"` and the system:

1. **Understands your query** via an AI agent
2. **Scrapes real LinkedIn profiles** using the Apify API
3. **Generates personalized sales messages** for each profile using a second AI agent
4. **Delivers results** back to you in Telegram and saves everything to Google Sheets

No manual prospecting. No copy-pasting. Pure automation.

## ⚙️ Workflow Architecture

```
Telegram Trigger
     │
     ▼
 AI Agent (Query Validator & Parser)
     │
     ├── ❌ Invalid Query → Error message to Telegram
     │
     └── ✅ Valid Query
              │
              ▼
        Apify LinkedIn Scraper (HTTP Request)
              │
              ├── 🚫 No Results → Friendly retry message
              │
              └── ✅ Profiles Found
                       │
                       ▼
                 JavaScript Processor
                 (Cleans & formats profiles)
                       │
                       ▼
                 AI Agent (Personalized Message Writer)
                 [Powered by OpenRouter]
                       │
                       ▼
              ┌────────────────────┐
              │  Google Sheets     │  ← Leads saved
              │  Telegram Reply    │  ← Message + LinkedIn URL
              └────────────────────┘
```

## 🚀 Getting Started

### Prerequisites

- An **n8n** instance (self-hosted or cloud)
- A **Telegram Bot** token (from [@BotFather](https://t.me/BotFather))
- An **Apify** account + API key
- An **OpenRouter** API key
- A **Google Sheets** spreadsheet for lead storage

### Setup Steps

1. **Import the workflow** — Upload `linkedin_messages.json` into your n8n instance via *Workflows → Import from file*.

2. **Configure credentials:**
   - `Telegram` — Add your bot token
   - `Apify` — Add your API key in the HTTP Request header
   - `OpenRouter` — Add your API key in the LangChain node
   - `Google Sheets` — Connect your Google OAuth account

3. **Update the Google Sheet ID** — Replace the sheet URL in the *Append row* node with your own spreadsheet.

4. **Activate the workflow** — Toggle it on and open your Telegram bot.

## 🔒 Notes & Best Practices

- The workflow validates your query before making API calls to save Apify credits.
- Profiles with missing websites or emails are gracefully handled and skipped.
- Each lead's data is normalized via a JavaScript node before AI processing.
- Messages are generated by an AI persona acting as a sales expert, tailored to each profile.
